  # Data Structures using GDB

## Overview

This repository contains implementations of basic data structures in C:

* Queue
* Stack
* Linked List

The focus of this project is to understand how these data structures work internally by analyzing their execution using GDB (GNU Debugger).

---

## Objectives

* Implement core data structures in C
* Observe their behavior step-by-step using GDB
* Understand memory usage and control flow

---

## Data Structures Included

### Queue

* Array-based implementation
* FIFO (First In First Out)

### Stack

* Array-based implementation
* LIFO (Last In First Out)

### Linked List

* Dynamic memory allocation
* Node-based structure

---

## How to Run

### Compile

```
gcc -g filename.c -o output
```

### Run with GDB

```
gdb ./output
```

---

## GDB Commands

* `run` → start execution
* `next` → execute next line
* `step` → enter function
* `print variable` → inspect variable

---

## Key Learning Outcomes

* Understanding of stack, queue, and linked list operations
* Visualization of execution using GDB
* Debugging and tracing program flow
